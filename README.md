## Project info

#### Main technologies used
- Java 11
- Spring Boot
- Docker
- Swagger
- Unit Test

### Clone project in your machine and enter the project folder

#### Copy and run this command bellow

```sh
mvn clean && mvn install  && sudo docker build -t eli/grao-api:0.0.1 . && sudo docker-compose up
```
